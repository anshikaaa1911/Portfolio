# Anshika Bharti Portfolio

This portfolio showcases my journey as a Computer Science undergraduate specializing in Artificial Intelligence and Data Science. It highlights my work in machine learning, data analytics, full-stack development, recommendation systems, cybersecurity awareness, and intelligent software solutions.

Through this portfolio, I present my technical skills, featured projects, certifications, co-curricular involvement, and professional experience. The website is built as a simple static portfolio using HTML, CSS, and JavaScript, making it lightweight, responsive, and easy to deploy on GitHub Pages.

## Files

- `index.html` - portfolio structure and content
- `styles.css` - page styling and responsive layout
- `script.js` - section navigation and dynamic content rendering
- `public/resume.pdf` - resume linked from the navigation

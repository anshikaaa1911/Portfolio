#  Anshika Bharti- PORTFOLIO

A modern and responsive portfolio website showcasing my academic journey, technical expertise, projects, certifications, leadership experiences, and professional growth as a **Computer Science undergraduate specializing in Artificial Intelligence and Data Science**.

This portfolio serves as a centralized platform for recruiters, hiring managers, and collaborators to explore my work in **Artificial Intelligence, Machine Learning, Data Analytics, Full-Stack Development, Cybersecurity, Recommendation Systems, and Intelligent Software Solutions**. It highlights both my technical capabilities and my passion for developing innovative, real-world applications that create meaningful impact.

Built using **HTML, CSS, and JavaScript**, the website is lightweight, fast, fully responsive, and easily deployable on static hosting platforms such as GitHub Pages.

---

##  Features

- Professional introduction and career overview
- Technical skills organized by domain
- Featured projects with technology stack details
- Certifications and achievements
- Education and academic background
- Leadership and extracurricular involvement
- Downloadable resume
- Fully responsive design for desktop, tablet, and mobile devices
- Lightweight architecture with no external framework dependencies
